Навигатор по туннелям

Это интерактивный путеводитель по системе транслокаторов. Сначала идет общий путь до развилки. Затем выберите нужное направление, чтобы увидеть дальнейший маршрут.
Как использовать: Нажмите кнопку "Копировать" рядом с командой, затем вставьте ее в чат игры (Ctrl+V).

Часть 1: Путь от спавна до развилки

Этот маршрут общий для всех направлений. Следуйте по транслокаторам от 1 до 38.

Транслокатор 1 / 38

/waypoint addati spiral 3967 110 14189 false #FF0000 Транслокатор к развилке (1/38)

Транслокатор 2 / 38

/waypoint addati spiral 9258 110 20163 false #FF0000 Транслокатор к развилке (2/38)

Транслокатор 38 / 38 (Выход на развилку)

/waypoint addati spiral 127455 110 155253 false #FF0000 Транслокатор к развилке (38/38)

Часть 2: Развилка

Вы достигли основной развилки. Теперь выберите, куда вы хотите отправиться дальше.
